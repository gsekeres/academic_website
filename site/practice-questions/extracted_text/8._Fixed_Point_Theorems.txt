1      Recap
A fixed point of a self-mapping function f : X ‚Üí X is a point x ‚ąą X such that f (x) = x. For
example, if X = [0, 1], fixed points are all the points in the image of f that lie on the 45 degree
line. We have already seen a fixed point theorem when you covered Euclidean topology.

Proposition. Suppose f : [0, 1] ‚Üí [0, 1] is continuous. Then, f has a fixed point.

     We present a number of fixed point theory that are commonly used in economics; e.g., proving
existence of equilibrium in games. With a notable exception of the Contraction Mapping Theorem,
most fixed point theory tells us that there is a solution, but not how to obtain that solution.
