Template Notes

I modified a template from Zahra Thabet, to whom I am grateful. I adjusted the template to compile correctly in Texifier using MacTeX, including local icon files rather than relying on the Font Awesome package.

The PDF is the canonical version. The TeX files are included because templates are easier to adapt when the source is visible.
